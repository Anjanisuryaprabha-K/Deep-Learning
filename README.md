# 🚀 Deep Learning Programs Repository

This repository contains a comprehensive collection of **Deep Learning projects** implemented using **Python** and **TensorFlow/Keras**. It showcases practical applications of key deep learning concepts including **Neural Networks, CNNs, RNNs, and NLP techniques**.

---

## 📌 Overview

The goal of this repository is to provide hands-on implementations of deep learning models across different domains such as:

- Computer Vision
- Natural Language Processing (NLP)
- Predictive Modeling

Each project is designed to be beginner-friendly, well-structured, and easy to understand.

---

## 📂 Projects Included

### 🧠 Neural Networks

- **Handwritten Digit Recognition**
  - Classifies digits (0–9) using the MNIST dataset

- **House Price Prediction**
  - Predicts house prices based on input features

---

### 📝 Natural Language Processing (NLP)

- **Movie Review Sentiment Classification**
  - Classifies reviews as positive or negative using the IMDB dataset

- **Newswire Topic Classification**
  - Categorizes news articles into topics using the Reuters dataset

- **One-Hot Encoding**
  - Converts categorical/text data into numerical format

- **Word Embeddings**
  - Represents words as dense vectors for NLP tasks

- **Recurrent Neural Network (RNN)**
  - Handles sequential data like text

---

### 🖼️ Computer Vision (CNN)

- **CNN Digit Classification**
  - Uses CNN for handwritten digit recognition

- **Dog vs Cat Classification**
  - Classifies images of dogs and cats

- **CNN Image Classification**
  - General image classification using convolutional neural networks

---
