Deep Learning Programs
This repository contains a collection of Deep Learning programs implemented using Python and TensorFlow/Keras. The programs demonstrate various deep learning concepts including neural networks, convolutional neural networks (CNN), recurrent neural networks (RNN), and natural language processing techniques.

Programs Included
Handwritten Digit Recognition Implementation of a neural network to recognize handwritten digits using the MNIST dataset.

Movie Review Sentiment Classification Deep learning model to classify movie reviews as positive or negative using the IMDB dataset.

Newswire Topic Classification Model that categorizes news articles into different topics using the Reuters dataset.

House Price Prediction Neural network model used to predict house prices based on different housing features.

CNN Digit Classification Convolutional Neural Network used for handwritten digit recognition.

Dog and Cat Image Classification CNN-based model that classifies images of dogs and cats.

CNN Image Classification Implementation of image classification using convolutional neural networks.

One Hot Encoding Demonstration of converting categorical or text data into numerical format using one-hot encoding.

Word Embeddings Implementation of word embedding techniques for representing words as dense vectors.

Recurrent Neural Network (RNN) Implementation of an RNN model for handling sequential data such as text.
